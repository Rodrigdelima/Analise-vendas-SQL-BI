# Power BI – Dashboard de Análise de Vendas

Este dashboard foi desenvolvido a partir de uma VIEW consolidada em SQL, criada especificamente para consumo no Power BI.

## Objetivo
Apresentar uma visão executiva do desempenho de vendas, permitindo análise rápida de faturamento, lucro e eficiência comercial.

## KPIs apresentados
- Faturamento Total
- Lucro Total
- Margem de Lucro (%)
- Quantidade Vendida
- Total de Pedidos
- Ticket Médio

## Análises Visuais
- Evolução do faturamento mensal
- Comparação de faturamento por região

## Tecnologias
- Power BI
- DAX
- PostgreSQL (camada de dados)

  dashboard reage automaticamente a filtros e à atualização da base de dados.
